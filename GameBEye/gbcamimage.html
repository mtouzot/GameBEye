GameBEye.gbcamimage package

The GameBEye.gbcamimage package is as its name indicates focused on Game Boy Camera image. Each image has a 128x112 pixels resolution and uses a 4-color palette (grayscale values, natively).

class GameBEye.gbcamimage.gbcamimage.GBCamImage

The base class of Game Boy Camera images.

A Game Boy Camera image is a nothing but a 4-colors 128x112 array.
